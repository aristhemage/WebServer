User Guide:
1.	Save the two files of C code ( server.py and client.py )
2.	Open two command prompt terminals (A will be the server and B with be the client).
3.	Change directory for both A and B to the directory/location of the code you saved in step 1:
cd <directory of server.py / client.py here>
4.	Run in terminal A – the server:
python server.py
5.	Run in terminal A – the server:
python client.py
